<h2 id="5">자기소개서</h2>
<p><strong>디자인부터 데이터베이스 연동까지 비즈니스 가치를 실현하는 5년 차 프론트엔드 개발자</strong><br/> 
  지난 5년 동안 프론트엔드 개발자 및 퍼블리셔로서 랜딩페이지 제작, 반응형 웹 구축, 서비스 유지보수 등 다양한 프로젝트를 성공적으로 수행해 왔습니다. HTML5, CSS3, jQuery, PHP, JavaScript를 기반으로 사용자 경험을 고려한 최적의 솔루션을 제공하며, 특히 마감 기한을 엄수하는 뛰어난 생산성과 문제 해결 능력을 갖추고 있습니다. 단순히 화면을 구현하는 단계에 머물지 않고 서비스 전반의 메커니즘을 이해하며 백엔드와 원활히 소통하는 든든한 협업 파트너가 되고자 합니다.</p>
<p><strong>디자인 감각과 백엔드 이해도를 갖춘 넓은 기술 스펙트럼</strong> <br/>  
  기술적으로는 프론트엔드의 핵심인 HTML5, CSS3, JavaScript를 능숙하게 다루며 Ajax와 Vue.js를 활용한 인터랙션 구현이 가능합니다. 백엔드 영역에서는 PHP와 CodeIgniter 환경에서의 개발 경험이 있으며, MySQL을 활용한 데이터 관리 능력과 SQLD 자격을 갖추고 있습니다. 또한 Git과 SVN을 통한 버전 관리, AWS S3 및 소셜 로그인 API 연동 등 인프라와 외부 서비스 결합에도 익숙합니다. 웹디자인기능사와 GTQ 1급 자격을 바탕으로 포토샵과 일러스트레이터를 활용한 디자인 작업까지 유연하게 수행할 수 있는 것이 저만의 차별화된 강점입니다.</p>
<p><strong>㈜티앤제이비즈 - 광고 성과 극대화를 위한 기동력과 최적화 솔루션 제공</strong><br/>
최근 ㈜티앤제이비즈에서는 대리로 재직하며 광고 성과 극대화에 특화된 랜딩페이지 제작을 전담했습니다. 상조, 보험, 대출 등 다양한 업종의 기획안을 바탕으로 디자인부터 퍼블리싱, 매체별 배너 제작까지 전 과정을 통합적으로 관리했습니다. 특히 모바일 유입이 높은 비즈니스 특성을 고려해 최적화된 인터페이스를 설계함으로써 이탈률을 낮추고 마케팅 효율을 높이는 데 기여했습니다. 이외에도 자사 홈페이지 리뉴얼, 외부 API 연동을 통한 데이터 전송 프로세스 구축, AI 기술을 활용한 영상 제작 지원 등 전방위적인 기술 지원 역량을 발휘했습니다.</p>
<p><strong>㈜마이패스 및 모노얼라이언스 - 안정적인 서비스 구축과 기술적 유연성의 증명</strong><br/>  
이전 ㈜마이패스에서는 주임으로 근무하며 기존 홈페이지를 모바일 반응형으로 전면 개편하는 성과를 거두었습니다. 대형 사이트를 단 2주 만에 리뉴얼하며 기기 환경에 제약 없는 사용자 경험을 구현했고, AWS S3 연동 및 카카오 알림톡 등 비즈니스 필수 기능을 안정적으로 결합했습니다. 또한 안드로이드 웹뷰 앱 개발과 소셜 로그인 연동을 스스로 학습하여 완수하며 기술적 유연함을 증명했습니다. 커리어의 시작이었던 모노얼라이언스에서는 한국기술교육대학교와 한국투자금융지주의 프로젝트에 참여하여 플래시 콘텐츠의 웹 표준 전환 및 크로스 브라우징 최적화를 완수하며 실무 기초를 탄탄히 다졌습니다.</p>
