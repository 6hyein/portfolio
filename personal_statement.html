자기소개서

정해진 마감 안에 작업을 끝낼 수 있도록 노력합니다. 어릴 때부터 시간 약속을 중요시하시는 부모님 곁에서 자라서 약속한 시각을 중요하게 여깁니다. 혹시라도 개인적인 역량이 부족해 시간 안에 일을 해결 수 없는 경우엔 검색이나 강의들을 참고해 조사했습니다. 이른 시일 내 해결할 수 없을 것 같은 경우 미리 상황을 보고드려 함께 해결책, 대안을 찾아 해결했습니다. 업무 과다로 마감 안에 끝낼 수 없을 것 같은 경우엔 미리 현 상황을 보고드리고 중요한 순서대로 업무를 먼저 처리해 회사일정에 지장이 없도록 노력했습니다.

더조은컴퓨터 아카데미에서 웹디자인(웹 퍼블리셔) 프런트 앤드 실무자 양성과정을 들었습니다. 디자인 수업으로 일러스트레이터, 포토샵을 배웠습니다. 프런트 앤드 수업으로 HTML5, CSS3, jQuery를 주로 배우고 JavaScript, Vue.js는 간단하게 배웠습니다. 수업하면서 디자인보다는 오류를 찾고 해결하는 과정이 있는 코딩이 작업에 흥미를 느꼈습니다.

모노얼라인언스 회사에서 5개월 계약직으로 근무했습니다.  SVN을 사용해 협동 작업을 하는 회사였습니다. 한국기술교육대학교의 플래시 학습파일의 소스를 웹사이트에 운용할 수 있도록 변환하는 작업을 팀과 나누어 했습니다. 한국 투자금융 지주 사이트를 함께 팀으로 작업했습니다. 혼자 작업하던 포트폴리오 작업에서 벗어나 동료와 함께 서로 모르는 것은 물어보며 작업하는 과정이 즐거웠습니다. 사수분의 지도로 효율적이며 정확하게 작업하는 방법을 배울 수 있었습니다.

마이패스 회사에서 2년 이상 정규직으로 근무했습니다. 새로운 언어들을 많이 접해 볼 수 있었습니다. 입사 후 생활코딩 PHP + mySQL 책과 강의를 통해 사이트에 주로 사용하는 언어를 익혀 작업했습니다. Codelgniter기반 CI보드를 사이트에서 사용하고 있어서 두 언어를 일하며 많이 사용했습니다. 이외 파이썬, JAVA 기초도 학습했습니다. 여러 언어를 접해보니 기본적인 부분에선 비슷한 점이 많아 신기했습니다.

마이패스 근무를 하면서 처음부터 도움 없이 스스로 작업해보는 경험을 많이 했습니다. Ajax를 이용해 데이터를 넘겨보기도 하고, 안드로이드 웹뷰 앱 개발, 소셜로그인 연동, AWS S3 버킷을 사이트와 연동해 파일 업로드, 카카오톡 템플릿 이용해 알림 보내기 등. 처음 접해보는 작업을 혼자 작업해 보는 경험을 해봤습니다. 할 수 없을 것 같던 업무들을 검색을 통해 해결하면서 새로운 것에 겁내지 않는 자신감을 배울 수 있었습니다.

포트폴리오 모임에 들어가서 MBTI 단톡방 과일 TEST 사이트를 웹 프론트엔드로 참여해 작업했습니다. 디자이너 2명과 개발자 2명으로 협동작업을 했습니다. 전체와는 노션으로, 개발자와는 GitHub를 이용해 작업을 공유했습니다. GitHub를 처음 사용해봐서 낯설었지만 주로 사용하는 명령어를 메모해 작성하며 익숙해졌습니다. 이해되지 않는 부분이 있으면 게더타운, 오프라인 만남을 통해 소통하며 맞춰 작업했습니다. 비슷한 연령대의 사람과 함께 이견을 조율하며 작업하는 순간이 즐거웠습니다.

지원하게 된 동기는 함께 팀을 이뤄 작업하는 즐거움을 더 느끼고 싶어서입니다. 앞서 경험해 본 포트폴리오 모임을 통해서 한 프로젝트를 위해서 함께 분업하기도 하고 서로의 작업을 보기도 하는 협동작업에서 즐거움을 느꼈습니다. 이전 회사 마이패스는 중소기업의 회사여서 함께 팀을 이뤄 작업하는 경험을 하기 힘들었습니다. 경험이 부족한 협동과 소통의 부족함을 채우기 위해 수영 소모임에 참여해 팀 핀추발 대회도 나가며 해당 역량을 키우려 노력하고 있습니다.
